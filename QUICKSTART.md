## Troubleshooting

### "Python version too old"
- **Solution**: Upgrade to Python 3.10 or higher
- Check version: `python --version`

### "Module not found" error
- **Solution**: `pip install -r requirements.txt`

### Lost your progress?
- **Don't worry**: Check for `save_data.json.bak`
- Rename it to `save_data.json` to restore

### Want to start fresh?
- Press `X` at the main menu to reset progress
- Or delete `save_data.json`
